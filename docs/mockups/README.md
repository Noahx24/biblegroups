# Marketing mockups

Drop these 5 PNG files in this folder. The marketing page references them by
exact filename:

| Filename       | What goes there                                                        |
|----------------|------------------------------------------------------------------------|
| `schedule.png` | The "No more group-chat chaos" Schedule mockup (mobile)                |
| `groups.png`   | The "Your church family, organised" Groups mockup (mobile)             |
| `family.png`   | The "Children's programmes, parental peace of mind" Family mockup      |
| `hymns.png`    | The "Sing along, in your language" Hymn Book mockup (mobile)           |
| `admin.png`    | The "Run the church, from the web" admin portal Dashboard (desktop)    |

## Recommended dimensions

- Mobile mockups: roughly 1080 × 1350 (4:5) - they will render at the grid
  cell width and scale down.
- Admin mockup: 16:10 or 16:9 - rendered up to 920px wide on desktop.

Anything PNG will work; the page constrains width and preserves aspect ratio.
For best Retina sharpness, export at 2x the rendered size.
